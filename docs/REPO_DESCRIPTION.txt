CEREBRUM is an autonomous, training-free Knowledge Graph reasoning engine that reasons like a Transformer—using community structure as attention, path-depth as layer-depth, and topology as the model. No LLM, no training data, and 100% glass-box interpretability. v2.51.0 (Phase 167) features zero-config GraphProfiler, Semantic Terminal Relation Boost (STRB), and Hybrid-Memory Mmap storage.
